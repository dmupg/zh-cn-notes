# zh-cn-notes

## 简介

本仓库用于归档和发布多个独立的 HTML 页面。这些页面内容以中文为主，涵盖技术笔记、文档整理、学习记录等用途。每个页面均为独立文件，不依赖特定域名或网站环境，可直接在浏览器中打开查看。

## 目录说明

```
zh-cn-notes/
├── README.md          # 本文件
├── index.html         # 首页/目录页（可选）
└── pages/             # 存放各独立 HTML 页面
    ├── note-1.html
    ├── note-2.html
    └── ...
```

- `pages/` 目录下存放所有归档的 HTML 页面，文件名以有意义的简短英文或拼音命名。
- 若存在 `index.html`，可作为导航页或列表页，方便浏览。

## 页面归档说明

- 每个 HTML 文件均为独立页面，包含完整的文档结构与样式。
- 页面内容可能包含代码片段、图表、链接或多媒体资源，但均嵌入或同目录引用，不依赖外部服务。
- 归档内容可能不定期更新或补充，不保证所有页面长期保持最新。
